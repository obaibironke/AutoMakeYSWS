This is a website for the YSWS Automake. Automake is a YSWS where you build an automation that solves a real problem, get an SBC to host it. You can see the live website at automake.dino.icu.

This website uses:
React and Typescript for the frontend UI
Vite as a build tool
TailwindCSS for styling
Framer Motion for animations
Wouter for client side routing
Vercel to host the website and use serverless functions
Airtable as a database for all sorts of participant information
Hack Club Auth for sign in and for projects to be attributed to a user

This website features a landing page, some guides, and a project showcase (which is currently empty) and an admin panel for reviewers that is only accessible to me and Dev. Dev if you are reading this the admin page is only accessible if you sign into the website seperately first and then go to automake.dino.icu/admin. Additionaly there is a working auth system with a user dashboard and shop that are only available after sign in.

This website uses some sensitive keys that are stored as environment variables and only accessable by me.
