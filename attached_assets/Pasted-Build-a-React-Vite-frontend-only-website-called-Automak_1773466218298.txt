Build a React + Vite frontend-only website called **Automake YSWS** from scratch. No backend, no API, no auth, no database, no monorepo, no workspaces.

---

## LAYOUT & DESIGN INSPIRATION

Model the layout, feel, and structure closely after https://blueprint.hackclub.com. Specifically:
- Big, bold hero section with a large program logo/title, tagline, and CTA buttons
- "Here's How It Works" section with icon + step cards in a horizontal row
- A scrolling marquee/carousel strip showing participant project cards (name, age, location style)
- A "Join our community" style section with image + text side by side
- A "Already a pro?" or showcase section with a grid of project cards linking to GitHub/details
- An FAQ accordion section at the bottom
- A rich footer with multi-column links, program description, and nonprofit-style copy
- Sticky navbar at top with logo left, nav links right
- Full-width alternating sections with generous padding
- Playful but clean — uses big images, bold type, flag/divider SVG decorations between sections

Adapt all of this to Automake YSWS content and branding (not Hack Club branding).

## BRANDING

**CSS variables in `client/src/index.css`:**
```css
:root {
  --eucalyptus: #D1DCCF;
  --midnight-plum: #3B2F3E;
  --pencil-lead: #424242;
}
```

**Google Fonts** (import in index.css):
- `DM Serif Display` — all headings
- `DM Sans` — all body text

**Design rules:**
- Hero and alternating sections: Eucalyptus (`#D1DCCF`) background
- Headings, all buttons, CTAs, logo text: Midnight Plum (`#3B2F3E`)
- Body text, borders, card details: Pencil Lead Grey (`#424242`)
- All buttons: Midnight Plum bg, white text, rounded-lg, px-6 py-3
- Cards: white bg, soft shadow, Midnight Plum border + lift on hover
- Sections alternate: Eucalyptus → white → Eucalyptus → white
- Decorative SVG dividers between major sections (wavy or flag-style, like blueprint.hackclub.com)
- Sticky navbar, full-width sections, generous vertical padding
- Fully mobile responsive — 1 col mobile → 2 col tablet → 3 col desktop

## PAGES & CONTENT

### Landing (`/`)

**Navbar** — sticky, white bg:
- Left: "Automake" logo text in Midnight Plum, DM Serif Display
- Right: nav links — Showcase, Guides, Shop — in DM Sans
- Mobile: hamburger menu

**Hero section** (Eucalyptus bg, centered):
- Large program title: "Automake YSWS"
- Tagline: "Build automation projects. Earn currency. Unlock rewards."
- Two CTA buttons: "Explore Projects" → /showcase, "Browse Guides" → /guides
- Large illustration or placeholder image below (use a colorful SVG placeholder)

**How It Works** (white bg):
- Section heading: "Here's How It Works"
- 4 horizontal step cards with icon + label:
  1. 🔧 Build an automation project
  2. 📋 Log your hours & submit
  3. ✅ Get reviewed & approved
  4. 🪙 Earn currency & shop rewards
- CTA below: "Follow the Guides →" linking to /guides

**Marquee strip** (Midnight Plum bg):
- Continuously scrolling horizontal strip of participant cards (CSS animation, no JS library)
- Each card: name, age, location — e.g. "Amara · 16 · Nigeria", "Jake · 15 · USA"
- Hardcode ~12 participant entries, loop them

**Featured Projects** (Eucalyptus bg):
- Heading: "Projects shipped so far"
- Grid of 3 project cards pulled from projects.ts
- "See all projects →" link to /showcase

**Join the Community** (white bg, image left + text right):
- Heading: "Join a community of young builders"
- Body: "Automake YSWS is a program for teens who want to build real automation projects and get rewarded for it. Submit your work, earn currency, and unlock tools, gadgets, and more."
- CTA button: "Get Started" → /guides

**FAQ accordion** (Eucalyptus bg):
- Heading: "FAQ"
- 6 hardcoded questions with expand/collapse toggle (pure React state, no library):
  1. Who can participate? — Any teen aged 13–18 anywhere in the world.
  2. How do I earn currency? — Submit an automation project, log your build hours, and get approved by an organizer.
  3. What can I buy in the shop? — Automation tools, tech gadgets, fun items, learning resources, and high-tier milestone rewards.
  4. Do I need prior experience? — Not at all! Starter guides are available to help you build your first project.
  5. How many projects can I submit? — As many as you like! Each approved project earns you currency.
  6. When does the program end? — Check back for updates — the program runs on a rolling basis.

**Footer** (Midnight Plum bg, white text, multi-column):
- Left: "Automake YSWS" logo + 1-line description
- Columns: Program links (Showcase, Guides, Shop), Resources (FAQ, About), Community
- Bottom bar: "A program for teen builders."

- Video URL placeholder
- Full documentation text
- "← Back to Showcase" link

### Guides (`/guides`)

- Page hero: heading "Starter Guides", subheading explaining guides are starting points to remix
- Filter pills: All / Beginner / Intermediate / Advanced
- Grid of ~6 guide cards from guides.ts: title, difficulty badge (green/yellow/orange), description, "View Guide →"

### Guide Detail (`/guides/:id`)

- Title, difficulty badge
- Intro paragraph
- Numbered step-by-step instructions
- "Ways to Make It Your Own" section — bullet list of modification prompts
- Example project cards at bottom

### Shop (`/shop`)

- Page hero: heading "Shop", note: "Earn currency by submitting approved projects, then spend it here"
- Category filter pills: All / Automation Tools / Tech & Gadgets / Fun & Quirky / Learning / Milestone Rewards
- Grid of item cards from shopItems.ts: name, category badge, description, cost (🪙 + number in bold), greyed-out "Locked" button (visual only, not interactive)

### 404 (`*`)

- Friendly message, link back home
