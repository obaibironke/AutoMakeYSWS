- Use a `useRef` on the section and a `useEffect` with a `scroll` event listener on the snap scroll container div (give it an id of "scroll-container")
- As the user scrolls DOWN, translate the entire logo field UP (negative Y) — as the user scrolls UP, translate the logo field DOWN (positive Y)

- Apply the transform to a wrapper div that contains all logo cards using `ref.current.style.transform = translateY(...)`
- Use `will-change: transform` and `transform: translateZ(0)` on the wrapper for GPU acceleration
- Make sure the parallax wrapper overflows the section bounds slightly (add overflow: hidden to the section, let the wrapper be position: absolute, inset: -10% 0)
